# WSL-jitsi<br />
WSL2 jitsi-meet install script (Install jisti-meet under Windows 10)<br />
Usage <br />
1.Install WSL2 (在Windows 10 2004以上或1903以上安裝WSL 2)<br />
https://docs.microsoft.com/zh-tw/windows/wsl/install-manual<br />
2.Get Jitsi meet GPG Key (取得Jitsi Meet的GPG金鑰)<br />
https://download.jitsi.org/jitsi-key.gpg.key<br />
3.Put GPG and install-jitsi.sh in C:\Users\(Username) (把前面的GPG金鑰和安裝用的sh放在Windows的使用者目錄下)<br />
4.win+R cmd.exe(or powershell) and run bash (用cmd下命令bash啟動linux)<br />
5.sudo sh install-jitsi.sh (用sudo執行腳本)<br />
6.Do your jitsi setup and Done (設定IP完成 jitsi安裝)<br />
